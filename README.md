# AI Agents Lab

A collection of AI agent projects — from beginner to advanced — built to learn, experiment, and showcase agent development patterns.

## Structure

```
ai-agents-lab/
├── shared/              # Common utilities and tools shared across agents
│   ├── tools/
│   └── utils/
├── beginner/            # Simple, single-purpose agents
├── intermediate/        # Multi-tool and RAG-based agents
└── advanced/            # Multi-agent systems and autonomous agents
```

## Getting Started

```bash
# Clone the repo
git clone https://github.com/ashutoshdwivedi16/ai-agents-lab.git
cd ai-agents-lab

# Create a virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Navigate to any agent and follow its README
cd beginner/01-simple-chatbot
pip install -r requirements.txt
```

## Tech Stack

- **Language:** Python
- **LLM Providers:** OpenAI, Anthropic (Claude)
- **Frameworks:** LangChain, CrewAI, Claude Agent SDK (as needed per project)
